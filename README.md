# System Design Learning

A blended course in computer science fundamentals and system design, built lesson by lesson into three growing artifacts:

- **[`lessons/`](lessons/)** — every lesson as a complete markdown document, from "how the internet works" to full capstone designs.
- **[`labs/`](labs/)** — runnable, hands-on material: Docker Compose stacks, Python experiments, CLI walkthroughs, load tests. Free and local.
- **[`designs/`](designs/)** — the design portfolio: one folder per system (URL shortener, rate limiter, chat, news feed, …), each a complete design doc with architecture and sequence diagrams.

Plus two living reference files:

- **[`glossary.md`](glossary.md)** — every term ever introduced, in plain language.
- **[`toolkit.md`](toolkit.md)** — the latency numbers, estimation formulas, and decision tables of the trade.
- **[`troubleshooting.md`](troubleshooting.md)** — decodes the errors the labs may throw.
